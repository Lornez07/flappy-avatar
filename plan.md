## Key Design Decisions

### Game State Machine
```
IDLE → CROPPING → READY → PLAYING → GAMEOVER → (submit) → LEADERBOARD → READY
```

### Physics Properties
- `player.rotation`: Current angle (-20° on flap, max +70° on fall)
- `player.velocityY`: Vertical speed
- `player.rotationVelocity`: Rate of rotation change

### Canvas High-DPI Approach
- Render at `canvas.width * devicePixelRatio` internally
- CSS size stays at canvas.width (for responsive layout)
- Scales automatically on Retina/4K displays

### Supabase Schema
```sql
CREATE TABLE leaderboards (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP,
  player_name TEXT NOT NULL,
  score INT NOT NULL,
  avatar_url TEXT,
  UNIQUE(player_name, created_at)
);
```
